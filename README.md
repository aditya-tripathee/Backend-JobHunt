This is the **backend** of the Job Hunt Portal built using the **MERN Stack** (MongoDB, Express.js, React, Node.js).  
It provides RESTful APIs for managing **users, jobs, and applications**, and connects seamlessly with the frontend.

---

## 📌 Features

- User registration and login using JWT authentication  
- CRUD operations for jobs (Create, Read, Update, Delete)  
- Job applications by users  
- Admin functionality to manage jobs and applications  
- Secure routes with authentication middleware  
- RESTful API design for easy frontend integration  

---

## 🛠 Tech Stack

- **MongoDB** - Database  
- **Express.js** - Backend framework  
- **Node.js** - Runtime environment  
- **JWT** - Authentication & Authorization  
- **Mongoose** - MongoDB ODM  
- **Cors** - Cross-Origin requests  
- **Dotenv** - Environment variables management
